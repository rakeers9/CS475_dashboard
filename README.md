# StudyPulse

A student wearable wellness dashboard — built with React + Vite, Tailwind, and Recharts.

Surfaces biometric and activity data (recovery, sleep, HRV, stress, focus) in a way that's actionable during and after study sessions.

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

Built for CS475.
